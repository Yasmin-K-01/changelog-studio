# Release Notes - Version 1.1.0

We are excited to release **Version 1.1.0** of Changelog Studio. This release brings rich aesthetics, enhanced animations, and stable backend integrations.

### Key Highlights

*   **Glassmorphic Design:** Standardized the color palette with translucent elements and glowing neon accents.
*   **Performance Improvements:** Reduced rendering times for large Markdown files by 40%.
*   **Dynamic Markdown Rendering:** Full markdown support with syntax highlighting for code blocks.

### Changelog

*   [Added] Live markdown preview editor in the left control panel.
*   [Added] High-performance CSS spinner for refresh and loading states.
*   [Fixed] Content overflow issue on mobile devices.
*   [Fixed] Encoding issues when loading non-ASCII markdown files from disk.
*   [Security] Patched directory traversal vulnerability in file endpoints.
