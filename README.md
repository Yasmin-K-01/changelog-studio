# Changelog Studio 🚀

An elegant, autonomous, and production-ready Python Flask web application designed to parse, manage, and showcase dynamic project release notes with a stunning neon glassmorphic UI.

---

## ✨ Features

*   **Glassmorphic UI Design:** A beautiful dashboard with translucent elements, sleek borders, alternating row colors, and glowing neon accents.
*   **Dynamic Markdown Rendering:** Full markdown support with automatic parsing of release files (e.g., `v1.1.0.md`).
*   **Interactive Copy Button:** Code blocks automatically feature a floating "Copy" button on hover for quick clipboard actions.
*   **Consolidated CSV Export:** An "Export All (CSV)" utility that aggregates all local markdown updates and downloads a single `consolidated_changelog.csv` file.
*   **Live Simulation & Refresh:** Automated workspace refresh state indicators with sleek loading animations.

---

## 🛠️ Tech Stack

*   **Backend:** Python, Flask API
*   **Frontend:** HTML5, CSS3 (Glassmorphic variables), JavaScript (Vanilla)
*   **Environment Configuration:** YAML setup (`config.yaml`)
